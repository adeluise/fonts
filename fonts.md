## Download reference

Fonts with a `Download` value of `manual` require the user to download files from the URL column. All other download URLs point to GitHub directories containing `.woff2` files that can be fetched directly with `curl`.

### File manifest

**Uncut Sans** (variable):
- `UncutSans-Variable.woff2`

**Apfel Grotezk** (static, 4 weights):
- `ApfelGrotezk-Regular.woff2`
- `ApfelGrotezk-Mittel.woff2`
- `ApfelGrotezk-Fett.woff2`
- `ApfelGrotezk-Brukt.woff2`

**Necto Mono** (static, 1 weight):
- `NectoMono-Regular.woff2`
